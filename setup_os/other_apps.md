# neovim

## Ubuntu

## macOS

Use `bob` to install
```bash
bob use stable
```

## Configure

Expose configure
```bash
ln -s "${HOME}"/.vim "${HOME}"/.config/nvim
```

Open `nvim`, the packages are going to be installed automatically.

Install tree-sitters in neovim
```bash
:TSInstall python lua typescript javascript
```
